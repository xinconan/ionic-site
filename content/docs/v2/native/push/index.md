<div id="init"></div>
<h3><code>init(options)</code>
  
</h3>




Initialize the plugin on the native side.

```
var push = Push.init({
   android: {
       senderID: '12345679'
   },
   ios: {
       alert: 'true',
       badge: true,
       sound: 'false'
   },
   windows: {}
});
```

<h3><code>hasPermission()</code>
  
</h3>


Check whether the push notification permission has been granted.






<div class="return-value" markdown="1">

  <b>Returns:</b>
