<div id="show"></div>

<h3>
  <code>show()</code>
  

</h3>

Displays an InAppBrowser window that was opened hidden. Calling this has no effect
if the InAppBrowser was already visible.











<div id="close"></div>

<h3>
  <code>close()</code>
  

</h3>

Closes the InAppBrowser window.
