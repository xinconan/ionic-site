Toolbar

The toolbar is generic bar that sits above or below content. Unlike an Navbar, Toolbar can be used for a subheader as well.

Component

selector: ion-toolbar

Usage

<ion-toolbar>
  <ion-title>My Toolbar Title</ion-title>
</ion-toolbar>
