Toast

Improve this Doc Toast is a subtle notification that appears on top of an app's content. Typically, Toasts are displayed for a short duration of time than automatically dismiss. _For more information, Check out the [API docs](../api/components/toast/ToastController)._

Basic Usage

Demo Source ```typescript import { ToastController } from 'ionic-angular'; export class MyPage { constructor ( private toastController : ToastController ){ } presentToast() { let toast = this.toastController.create({ message: 'User was added successfully', duration: 3000 }); toast.present(); } } ```
