Toggle

Improve this Doc A toggle is an input component that holds a *boolean* value. Like the [checkbox](#checkbox), toggles are often used to allow the user to switch a setting on or off. Attributes like `value`, `disabled`, and `checked` can be applied to the toggle to control its behavior. _For more information, Check out the [API docs](../api/components/toggle/Toggle)._
