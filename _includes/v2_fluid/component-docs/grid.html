Grid

Improve this Doc Ionic's grid system is based on [flexbox](http://www.w3.org/TR/css3-flexbox/), a CSS feature supported by all devices that Ionic supports. The grid is composed of three units — _grid_, _rows_ and _columns_. Columns will expand to fill their row, and will resize to fit additional columns. Rows have no padding or margin applied to them. If that is needed, `ion-grid` can be added to provide additional padding. In order to set width use `width` attribute on ion-col tag. Demo Source ``` html This column will take 10% of space ```

Use the `offset` attribute on a column to set its percent offset from the left (eg: `offset-25`). To change how columns in a row align vertically, add the `center` or `baseline` attribute to an ``. Use the `wrap` attribute on an `` element to allow items in that row to wrap. This applies the `flex-wrap: wrap;` style to the `` element.
